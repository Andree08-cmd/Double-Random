Balanced FEN Dataset for Double Random Chess (960²)This repository contains an open-source Python script and a verified dataset of 445 balanced FENs for Double Random Chess (960²). The goal of this project is to eliminate severe first-turn imbalances and white/black asymmetries by using a rigorous, three-stage heuristic filtering method powered by Stockfish.The Filtering Methodology1. Stage 1: Initial Screening (Blunder Filter)Configuration: Depth = 12 | MultiPV = 1Cutoff Criteria: Evaluation must be strictly between $-1.0$ and $+1.0$.Technical Rationale: A depth of 12 plies equals an analysis of about 6 full moves ahead for each side. If Stockfish detects an advantage greater than 1.0 pawn of material in just 6 moves, the position has a severe structural imbalance. Positions that fail this check are discarded immediately to save computational time.2. Stage 2: Dynamic Options Refinement (The "Sweet Spot")Configuration: Depth = 23 | MultiPV = 5Cutoff Criteria: * The top 2 best moves must be evaluated between $-0.23$ and $+0.23$.The next 3 subsequent moves (MultiPV slots 3, 4, and 5) must be between $-0.8$ and $+0.8$.Technical Rationale: It is not enough for a position to be balanced in just one single engine line. Using MultiPV = 5 ensures that both players have multiple healthy, playable candidate moves. Limiting the top two choices near $0.0$ guarantees mathematical balance, while allowing the other three moves a margin of up to $\pm 0.8$ ensures the position remains rich in tactical resources and human choices, without instantly punishing a player for not finding the absolute "computer move".3. Stage 3: Strict Validation (Seal of Quality)Configuration: Depth = 30 | MultiPV = 5Cutoff Criteria: Final confirmation of the boundaries established in Stage 2.Technical Rationale: Depth 30 acts as the ultimate validation against the "horizon effect" (where the engine changes its mind upon looking deeper). If the stability and balance of the FEN hold steady under Master-level scrutiny, it is officially approved for competitive play.
